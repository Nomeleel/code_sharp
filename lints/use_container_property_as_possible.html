Using the Linter

Use Container property as possible.

A Container is combined common painting, positioning, and sizing widgets, so his 'child' can use its properties directly.

BAD:

Container(
  height: 77.0,
  width: 77.0,
  child: Center(
    child: FlutterLogo(),
  ),
)

GOOD:

Container(
  height: 77.0,
  width: 77.0,
  alignment: Alignment.center,
  child: FlutterLogo(),
)
