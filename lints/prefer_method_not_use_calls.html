Using the Linter

Prefer method direct calls instead of call calls.

BAD:


void fun() {}

void doSomething() {
  fun.call();
}

GOOD:


void fun() {}

void doSomething({VoidCallback? callback}) {
  fun();
  callback?.call();
}
